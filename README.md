## Features
- [x] login
- [X] registration
- [X] resend password
- [X] load cardsets/cards
- [X] show statistics
- [X] change settings
- [X] show cardcontent
- [X] learn-function (drag cards)
- [X] share cardsets
- [ ] format cardcontent (bold, italic, ...)
- [ ] push offline stats
- [ ] manage freindlist
